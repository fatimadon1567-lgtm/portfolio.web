## Features

- 3D animated particle network background (.js)
- Welcome splash screen with typewriter effect
- Glitch text animation on name
- Red / Cyan theme toggle (persists across pages)
- Hacker sound mode (Web Audio API — no files needed)
- Scroll reveal animations
- Mouse tilt effect on cards
- Fully responsive (mobile friendly)
- PHP contact form handler

---

## Tech Stack

`HTML5` · `CSS3` · `JavaScript` · `PHP` · `Python` · `.js`

## How to Run

Open `index.html` directly in any browser — no server needed for Home and About pages.

For the PHP contact form, run on a local server (XAMPP / WAMP) or deploy to any PHP host.
